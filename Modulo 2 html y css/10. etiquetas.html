Etiquetas HTML

html -> Contiene toda la página. Aquí va todo el contenido.

head -> Configuración. Aquí van cosas que no se ven como el título o enlaces a CSS.

body -> Lo que se ve. Todo lo visible para el usuario va aquí.

title -> Nombre de la pestaña. Es lo que aparece arriba en el navegador.

header -> Parte de arriba. Sirve para el título o logo de la página.

nav -> Menú. Se usa para los enlaces de navegación.

main -> Contenido principal. Aquí va lo más importante de la página.

section -> Secciones. Divide la página en partes.

article -> Contenido independiente. Como una noticia o publicación.

footer -> Parte final. Información como contacto o derechos.

div -> Agrupar cosas. No tiene significado, solo organiza.

p -> Texto. Sirve para escribir párrafos.

h1 -> Título grande. Es el más importante.

h2 -> Subtítulo. Va debajo del h1.

span -> Parte pequeña de texto. Se usa dentro de un texto.

a -> Enlaces. Sirve para ir a otra página usando href.

img -> Imágenes. Permite mostrar imágenes con src.

ul -> Lista con puntos. Para listas sin orden.

ol -> Lista numerada. Para pasos o orden.

li -> Elemento de lista. Cada punto de la lista.

form -> Formulario. Sirve para enviar datos del usuario.

input -> Escribir datos. Puede ser texto, contraseña, etc.

button -> Botón. Sirve para hacer clic y ejecutar acciones.

table -> Tabla. Organiza información en filas y columnas.

tr -> Fila. Representa una fila en la tabla.

td -> Dato. Es una celda dentro de la tabla.

video -> Video. Sirve para mostrar un video en la página.

audio -> Audio. Permite reproducir sonido.

source -> Fuente. Se usa dentro de video o audio para poner el archivo.

script -> JavaScript. Sirve para agregar funciones a la página.

link -> CSS. Se usa para conectar estilos a la página.
